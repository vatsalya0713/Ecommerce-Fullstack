# 🛒 Ecommerce-Fullstack Project

![Ecommerce Banner](https://img.shields.io/badge/Fullstack-Ecommerce-blue?style=for-the-badge)

A **Fullstack Ecommerce Application** with separate **Frontend (Ecomm-UI)** and **Backend (Ecomm-backend)** built using **Spring Boot, Thymeleaf, and MySQL**.

---

## 🌟 Features

- User-friendly UI for browsing products  
- Product management (CRUD) via backend  
- Connects frontend with backend using Spring Boot  
- Database management with MySQL  
- Responsive design for desktop and mobile
